## ✨ Features

* 📍 Select city in Syria (Damascus, Aleppo, Homs, etc.)
* 🕌 Display accurate daily prayer times:

  * **Fajr (الفجر)**
  * **Dhuhr (الظهر)**
  * **Asr (العصر)**
  * **Maghrib (المغرب)**
  * **Isha (العشاء)**
* ⚡ Real-time API data fetching
* 📱 Fully responsive design
* 🎯 Clean UI/UX experience
* 🔄 Dynamic content updates

---

## ⚙️ Technologies Used

* 🧱 HTML5
* 🎨 CSS3
* ⚡ JavaScript (ES6)
* 🔗 Axios (API Requests)

---

## 🌐 API Integration

This app uses the **AlAdhan Prayer Times API** to retrieve accurate prayer timings based on the selected city.
